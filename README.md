## 实验记录

### V1 手动辅助线引导 + 对话调整最终效果

!['辅助线引导'](./public/readme/layout-with-auxiliary-line.jpg)

- 根据经验手动绘制辅助线，示意定位，并将完整目标图作为输入，告知AI最终生成目标（但此刻暂停生成，仅作为参考信息录入）
- 根据经验截取局部，以组件为单位，要求AI根据图示布局生成相应代码
- 图片所不能直接暴露出的动态效果（例如横向滚动、纵向滚动等需求）作为补充信息录入
